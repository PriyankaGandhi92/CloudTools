Privacy Policy - BluePrint PDF Viewer Extension

Last Updated: May 11, 2026

Overview

The BluePrint PDF Viewer Chrome Extension does not collect, store, or transmit any personal data from users. This extension operates entirely on the user's device and does not send any information to external servers.

Data Collection

We do not collect:

  • Personally identifiable information (name, email, address, etc.)
  • Browsing history or web activity
  • PDF file contents (files are processed locally and redirected to the BluePrint editor)
  • Location data
  • Authentication credentials
  • Financial or payment information

Local Storage

The extension uses Chrome's local storage API solely to store the user's preference for enabling or disabling PDF interception. This is a single boolean value (true/false) that remains on the user's device and is never transmitted to any server.

Third-Party Services

This extension does not use any third-party services, analytics, or advertising networks. No data is shared with any third parties.

File Handling

When a user opens a PDF file through this extension:

  • Web PDFs: The extension redirects the URL to the BluePrint PDF Editor web application
  • Local PDFs: The extension reads the file locally and converts it to base64 to pass to the BluePrint editor
  • PDF contents are never stored or transmitted by the extension itself

User Rights

Users have the right to:

  • Disable or remove the extension at any time through Chrome's extension settings
  • Clear local storage data through Chrome's browser settings
  • Review the extension's source code (open source)

Changes to This Policy

Any changes to this privacy policy will be posted on this page with an updated revision date.

Contact

For questions about this privacy policy, please contact us through the BluePrint PDF Editor website.
